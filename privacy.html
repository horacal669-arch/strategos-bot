Política de Privacidad

1. Información que Recopilamos

Recopilamos:

  • Email y contraseña (encriptada)
  • WhatsApp (opcional, para soporte)
  • API Keys de Binance (encriptadas)
  • Datos de operaciones del bot
  • Información de pago (procesada por terceros: PayPal, Coinbase)

2. Uso de la Información

Usamos tus datos para:

  • Operar el bot en tu cuenta de Binance
  • Enviarte notificaciones de trading
  • Procesar pagos y suscripciones
  • Mejorar el servicio
  • Soporte técnico

3. Seguridad

Tus datos están protegidos con:

  • Encriptación SSL/TLS
  • Contraseñas hasheadas (SHA-256)
  • API Keys encriptadas en base de datos
  • Acceso restringido a servidores

4. Compartir Información

NO vendemos ni compartimos tus datos personales. Solo compartimos con:

  • Binance (para ejecutar trades)
  • Procesadores de pago (PayPal, Coinbase)
  • Telegram (para notificaciones)

5. Tus Derechos

Puedes:

  • Solicitar copia de tus datos
  • Eliminar tu cuenta en cualquier momento
  • Modificar tu información
  • Revocar API Keys

6. Cookies

Usamos cookies esenciales para mantener tu sesión. No usamos cookies de tracking.

7. Retención de Datos

Conservamos tus datos mientras tu cuenta esté activa. Después de eliminar tu cuenta, los datos se borran en 30 días.

8. Contacto

Para consultas sobre privacidad: support@strategos.hc
